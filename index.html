Spielregeln

Punktesystem

Joker = 0 · Ass = 1 · 2–K = 2–13

Vorbereitung

Karten mischen · Jeder erhält 5 Karten · Startwert: 0 Punkte

Ablauf

1. Der jüngste Spieler beginnt.

2. Karte ablegen (einzeln, Paare oder gleichfarbige Straßen), dann eine Karte ziehen.

3. Die Runde endet, wenn ein Spieler „Yaniv!" ruft (Handwert ≤ 7).

4. Alle anderen machen noch einen letzten Zug.

5. Assaf: Hat ein anderer Spieler ≤ Punkte wie der Yaniv-Rufer, erhält dieser Strafpunkte.

Besondere Regeln

Genau 75 Punkte → Reset auf 35 (bei Yaniv 200: 150 → 75).

Genau 100 Punkte → Reset auf 50 (bei Yaniv 200: 200 → 100).

Spielende

Ein Spieler verliert, wenn seine Punktzahl die Yaniv-Grenze überschreitet. Gewinner ist der Spieler mit den wenigsten Punkten.
